to run code make sure all libraries are downloaded 
to get the openai part to work must do pip install openai==0.28
then to get rest of code to work 
cd into directory
activate python enviorment using
python3 -m venv .env
next do
[Mac or Linux] source .env/bin/activate
[Windows] .env/Scripts/activate
and download the requirements
pip install -r requirements.txt
and make sure you add you openai key at top of code
